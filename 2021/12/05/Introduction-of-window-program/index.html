Introduction of window program

​ An introduction of window program

When I write codes, I always wonder that how others can use my code, especially for those who do not know much about Python. If they know little about Python, they even do not know how to run a single instruction. So I have to make my program more kind which means I have to improve the interactivity of my program. To achieve this purpose, I decide to introduce window program. In window program, there is button and displaying screen. By using this component, users can understand the purpose of the code better because tips can be written in the button or screen. For example, if I write a ‘plus 1’ on the button, users will know that if they click on the button, a number will plus 1. Here is an example. First, I will show my original codes

If you know little about Python, of course this is confusing. You will not know that to run this code you have to input ‘pickones()’ which is a fuction. But if I turn this code into window program.

It is more clear right? There is a button on the bottom of the left side called ‘pickone’, when users see this, they will know clicking the button can pick one. In another word, they know to run the code, they need to click the button. And the displaying screen is on the right of the button so that they can see the result.
