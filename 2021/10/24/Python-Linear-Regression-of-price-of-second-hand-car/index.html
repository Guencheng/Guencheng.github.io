Python--Linear Regression of price of second-hand car

This time, the task is about how to use regression to predict the price of second-hand car through some properties of the car. First of all, we should introduce some function to Python which has to been done every time. In addition, the dataset should be downloaded here.

This dataset has 31 rows including SaleID, 15 known properties and 15 unknown properties. Unknown properties are expressed by V.

Then, in the regression, SaleID and name of the cars are useless properties which should be dropped or the accuracy of the result may be affected.

There is another annoying point. The row of price is between the rows of known properties and the rows of unknown properties so we have to put price in the first row which is convenient for us to separate price and properties.

Next step is to separate test dataset and train dataset as well as price and properties. However, it is strange that unknown properties are all indicated by strings so we need to change the data type of unknown properties. The code with red line is what I did to solve this problem.

Finally, the most important step, use Linear Regression to train the data.

So far, all steps are done but we also need to test how well is the model trained. Here, I add the absolute difference of all trained value and all real value and then divide them by the total number. The final error is 2840 yuan.
