The preparation of X Academy

​ The preparation of X Academy

​ Before I went to X Academy, I had to do some preparation for this activity to make sure that I could understand what teachers would say. My program in X Academy was called Data Science and I divided my preparation work into three parts – basic Math abilities, basic machine learning knowledge and basic applications of Python.

​ First, I would discuss Math anilities. Among Math abilities, I also divided it into three parts—functions, linear algebra and statistic. The reason why I needed to learn all kinds of functions was because, in data analysis, we may need to use function to calculate some values such as entropy. Additionally, functions are helpful when we learnt some principles of a model. Then, it was about linear algebra which was the most difficult part in Math abilities. However, it was also essential as Many models are based on linear algebra. Statistic was the most important part because we must use statistic models when we analyzed data such as using normal distribution.

​ Secondly, it was about machine learning. I did not need to know it in detail because I would learn it in my program. In the stage of preparation, I just need to learn what each model is like.

​ Last, I would talk about Python. This is the most essential part among my preparation as I would use Python to analyze data in my program. If I know little about it, it would be hard for me to write or read codes, which may make me fall behind the class.
