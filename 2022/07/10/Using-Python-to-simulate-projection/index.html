Using Python to simulate projection

To realize this idea, I can only simulate it using python as, in real situations, there are many disturbing factors, such as air resistance. Also, I can just print all properties that I need of a projection at any fraction. The algorithm is easy to program. I just need to define the initial height, initial velocity and the angle of this velocity to the horizontal. Then, I wrote a algorithm to break this velocity into horizontal and vertical velocities. Also, I need to convert ‘distance is equal to speed multiplying time’ into algorithm so that I can get the coordinate of the object compared with initial position at any fraction of this projection.
