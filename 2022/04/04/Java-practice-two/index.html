Question: A person wants to buy some milk tea with 500 dollar and he wants to spend all of the money. How to write a code to find the solution that the man can buy as much milk tea as possible and there is no repetitive taste.

For this question, the first and the most tuff work for me is to input all data about milk tea which costs me a lot of time. Also, we need to define a new data type that can store the name of milk tea and its price together. Then, we have to order it with descending the price to make it more convenient for us to write code. I would just show part of the code to input data and the code to order milk tea.

Then, we need to find the solution to select most milk tea. To buy as much milk tea as possible, we need to buy more cheaper milk tea. My idea is to add all milk tea from cheapest to most expensive one by one until it first exceeds or is equal to 500 dollar. If it is 500 dollar, that is wonderful because there will not be any better solution but the real situation is that it exceeds 500 dollar. Then, the most important step comes. We should remove the last milk tea added as it is extra. After removing it, we can find that there is a gap between 500 dollar and our list so we should replace the last milk tea in our current list and we need a milk tea with a price which is equal to the gap plus the price of the milk tea we removed.
