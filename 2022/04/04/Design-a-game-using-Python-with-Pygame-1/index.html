Design a game using Python with Pygame - 1

Using Python to design a game

A few days ago, I found a teaching video about how to design a game using python in bilibili. I haven’t tried to do this so I decide to take a look.

This game is called ‘Plane fighting’. As the name suggests, in this game, there are two planes. One is enemy and one is me. Enemy is controlled by computer and our plane is controlled by ourselves. Both planes will shoot bullets. If we are hit, we will lose. If enemy is hit, we will get a point and then it will appear again util we lose. Because this code is a bit long and complex, I would explain step by step, I just give a general explanation of what I did. Also, I want to mention the whole game is designed using a library in Python called Pygame.

Step1: We should define a screen using pygame and all instructions we did will base on the screen. It is quite easy to do it – getting a picture used as screen then follow the instruction of pygame to display it.

Step 2: We should define a class called player. Inside the class, we need to concern two problems. One is to display the player. Second is how to control it. To solve the first question, we need to get a picture to represent the player and display it using functions contained in pygame. The same principle is applied to how to control it. There is a function in pygame to detect the information got from the keyboard.

Step3: We need to define a class called player_bullet. Bullet is different from player as bullet does not to be controlled so it is much easier to de defined.

Step4: This step is quite similar as the previous two steps. We need to define the classes for enemy and enemy bullets. The difference is that enemy does not to be controlled and it should follows a path.

Step5: We should write a program to detect whether player or enemy touches bullets.
