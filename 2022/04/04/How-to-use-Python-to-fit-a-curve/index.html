How to use Python to fit a curve

How to use python to fit a curve

This is a very meaningful topic as in engineering we might meet a lot of data and we need a curve to fit the data to get the trend. First, before we start we need a data to be fitted. Because I do not have data that can be used so I have to make up some data. I generate a data about the equation (x-2)(x-8)(x-5)+ 10. To make it more difficult, I will give some percentage error to it.

Then, with the data, we can start fit it. First, we need a model and I will use a library called curve_fit to help me. In this library, you just need to input the data and it will return the parameter you needed for you. With the parameter, we can draw a diagraph again to check whether it works. Here, I will the result.

The blue line is original data and the orange line is the data we fit. It is clearly shown that the library works well. For the blue line, there are some bends as we set percentage error. For the orange line, it is very smooth.
