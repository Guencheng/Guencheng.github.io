A timer -- combining hardware with software

Using Arduino to design a timer

Electronic engineering is a major that combines hardware and software. Hardware is the body of electronic engineering and software is the sprite of electronic engineering. I have not tried to combine them so far, so in this article, I will report a small timer designed by myself. The timer is made up of Arduino and it is coded by Arduino language. This report will be broken into three parts – design, hardware and software.

In the stage of design, we should get known what should be contained in this timer. There are two parts. First, there must be a screen to show the timer. Then, there must be components that can control the display of screen. The display of screen will be discussed in software part and the components needed will be contained in hardware part. I will introduce software part first.

Before coding, we need to think about what should be displayed in the screen. From my perspectives, there are 5 stages. One to tell you this is a timer.

​ One to ask you to set your time.

​ One to provide a platform for you to set the time.

​ One to display the time left.

​ One to tell you time is over.

That is all we need to consider and following, I will show my code to realize this. Also, I want to mention that the code not only control the display of screen it also controll the process of hardware.

​ The last part is about hardware. The major job of hardware is to convert our thoughts into the instructions to screen. Of all hardware, there are only two output components – screen and a buzzer. Why I need a buzzer here because user can not check the screen all the time so we need a component to tell the user that time is over. Other are input devices. For example, we need a stick to control the cursor on the screen. You can consider that the stick is a form of mouse used. Also, we need a button that can be clicked because mouse can be clicked but stick cannot so we need an extra component to realize this. In general, I divide mouse into stick and button in this project as Arduino does not provide mouse, at least I do not have mouse for Arduino.
